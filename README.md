# 🏥 Hospital Management & Medical Record System

## 📌 Project Description

This is a Spring Boot based backend application designed to manage hospital operations such as patient details, doctor information, medical records, prescriptions, and appointments.

The system follows a layered architecture to ensure clean code structure and scalability.

---

## 🚀 Features

* Patient Management (Create, Read, Update, Delete)
* Doctor & Department Management
* Medical Record Tracking
* Prescription Management
* Appointment Booking
* Exception Handling with proper error messages

---

## 🛠️ Tech Stack

* Java
* Spring Boot
* Hibernate / JPA
* MySQL / PostgreSQL
* REST APIs
* Postman (API Testing)

---

## 📂 Project Structure

* **Controller Layer** → Handles HTTP requests and responses
* **Service Layer** → Contains business logic
* **DAO Layer** → Handles database operations
* **Repository Layer** → JPA repository interfaces
* **Entity Layer** → Database models
* **DTO Layer** → Data Transfer Objects
* **Exception Handling** → Custom exception classes

---

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/Poojakk112/Hospital-Management-Medical-Record-System.git
   ```
2. Open the project in Eclipse / IntelliJ
3. Configure database in `application.properties`
4. Run the Spring Boot application
5. Use Postman to test APIs
